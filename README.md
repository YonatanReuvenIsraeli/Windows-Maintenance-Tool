# Windows Maintenance Tool v7.0.7
Perform various Windows miscellaneous tasks.

- Clear Windows Store cache
- Clean Windows Component Store.
  - Supports online and offline Windows installations.
  - Supports component and service pack.
  - Supports resetting or not resetting base.
- Restore default power plans.
- Clear auto-mount points.
- Reset OpenSSH client keys for current user.
- Clear Run history for current user.
